# 🚀 Enterprise Portfolio Pursuit Framework

**The Ultimate 6-Section Enterprise Pursuit Workbook**

Built for **Executive Landscaping, Inc.** with a fully loaded **Masonicare enterprise opportunity example**.

This project is an **interactive enterprise opportunity evaluation dashboard** built with **Streamlit** and packaged using **Docker** for fast deployment and portability.

---

# 📊 Features

- Interactive **6-section enterprise pursuit dashboard**
- **EOS Gate qualification scoring**
- **Enterprise financial modeling**
- **Property breakout analysis**
- **Competitive vendor displacement planning**
- **Operational readiness evaluation**
- **EOS execution tracker**
- **One-click Markdown report export**
- **Dockerized deployment**
- Ready for **future Salesforce integration**

---

# 🧠 Framework Sections

The application includes **7 main sections**:

1️⃣ **EOS Gate** – Enterprise qualification and scoring  
2️⃣ **Financial Model** – Annual opportunity financial modeling  
3️⃣ **Property Breakouts** – Territory and property analysis  
4️⃣ **Competitive Displacement** – Vendor replacement strategy  
5️⃣ **Operational Readiness** – Operational capacity and planning  
6️⃣ **EOS Tracker** – Enterprise execution tracking  
7️⃣ **Export Report** – Generate downloadable reports

# ⚡ Quick Start (90 Seconds)

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR-USERNAME/enterprise-portfolio-pursuit-framework.git
cd enterprise-portfolio-pursuit-framework
```

### 2. Run the Application

```bash
docker-compose up --build
```

### 3. Open in Browser

```
http://localhost:8501
```

---

# 🐳 Docker Deployment

This application is fully containerized using **Docker Compose**.

Start the application:

```bash
docker-compose up -d
```

Stop the application:

```bash
docker-compose down
```
